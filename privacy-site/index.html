ReCheck — Privacy Policy

Last updated: July 9, 2026

ReCheck runs entirely in your browser. It has no server, and the developer collects none of your data — no analytics, no tracking, no ads.

1. What ReCheck stores

ReCheck saves the following in chrome.storage.local — on your device only:

  • Your AI provider API key
  • Your settings: chosen provider, model, interface language, and number of questions

This never leaves your browser and is never sent to the developer.

2. What is sent, and to whom

When you click the “re-check” button, the text of the current page (or the portion you selected) is sent directly from your browser to the AI provider you chose, using your own API key. The developer operates no intermediary server and never sees this data.

The provider you select handles that request under its own privacy policy:

  • Anthropic (Claude): anthropic.com/legal/privacy
  • OpenAI (GPT): openai.com/policies/privacy-policy
  • Google (Gemini): policies.google.com/privacy

3. Page access

ReCheck reads the current page’s text only when you trigger it (by clicking the button or dragging a selection). It does not read or transmit page content automatically or in the background.

4. What the developer collects

Nothing. There is no server, no database, no analytics, and no third-party trackers. The developer cannot see your API key, the pages you read, or the questions generated.

5. Data retention and deletion

Your key and settings remain in your browser until you clear them in the extension, clear your browser data, or remove the extension. Removing the extension deletes them.

6. Children

ReCheck is a general-audience tool and is not directed at children under 13.

7. Changes

If this policy changes, the “Last updated” date above will change. Material changes will be reflected here before they take effect.

8. Contact

Questions about this policy: tmdwprpdla@naver.com

한국어

1. ReCheck가 저장하는 것

ReCheck는 아래 항목을 chrome.storage.local(사용자 기기)에만 저장합니다:

  • AI 제공자 API 키
  • 설정값: 선택한 제공자·모델·UI 언어·질문 수

이 값은 브라우저를 벗어나지 않으며 개발자에게 전송되지 않습니다.

2. 무엇이, 누구에게 전송되나

“다시 짚어보기” 버튼을 누르면 현재 페이지의 본문(또는 선택한 부분)이 사용자의 API 키와 함께 브라우저에서 직접 선택한 AI 제공자에게 전송됩니다. 개발자는 중계 서버를 두지 않으며 이 데이터를 볼 수 없습니다. 선택한 제공자는 각자의 개인정보 처리방침(위 링크)에 따라 처리합니다.

3. 페이지 접근

ReCheck는 사용자가 실행할 때만(버튼 클릭 또는 드래그 선택 시) 현재 페이지의 텍스트를 읽습니다. 자동으로 또는 백그라운드에서 페이지 내용을 읽거나 전송하지 않습니다.

4. 개발자가 수집하는 것

없습니다. 서버·데이터베이스·분석 도구·제3자 추적기가 전혀 없습니다. 개발자는 사용자의 API 키, 읽은 페이지, 생성된 질문을 볼 수 없습니다.

5. 데이터 보관 및 삭제

키와 설정은 사용자가 확장에서 지우거나, 브라우저 데이터를 삭제하거나, 확장을 제거할 때까지 브라우저에 남습니다. 확장을 제거하면 함께 삭제됩니다.

6. 아동

ReCheck는 일반 사용자용 도구이며 만 13세 미만 아동을 대상으로 하지 않습니다.

7. 변경
